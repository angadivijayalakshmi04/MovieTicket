A full-stack web application built using Spring Boot and Thymeleaf for seamless movie ticket booking.
This Movie Ticket Booking System allows users to browse movies, view show timings, book tickets, make online payments, and receive booking confirmation via email.
The application is built with Spring Boot, Thymeleaf, MySQL, and integrates Cloudinary, Razorpay, and QR Code Generation for a full end-to-end booking flow.
